- `withArrayCopy()` 리팩토링 순서가 약간 이상. 리팩토링 중간에 코드를 깨먹는다.
- `ES2023에는 Array.prototype.with()`가 추가됨

```javascript
const arr = ['a', 'b', 'c', 'd', 'e'];
console.log(arr.with(2, 'f')); // ['a', 'b', 'f', 'd', 'e']
console.log(arr); // 'a', 'b', 'c', 'd', 'e'
```
